## 桜の接ぎ木
## 接木之櫻

在這裡<br>
我們將會把我們團隊喜歡的日文歌詞翻譯成中文<br>
並與大家分享日文歌詞中的意境 以及其表現的細緻  
  
就如將日文的櫻花嫁接到中文的樹上一般<br>
目的是希望能讓大家用中文欣賞日文歌詞的美<br>
因而取名為「桜の接ぎ木」（接木之櫻）
